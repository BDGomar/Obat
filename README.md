# Obat Services - Site Web React

Site web moderne pour Obat Services, converti depuis une maquette HTML en application React.

## Installation

```bash
npm install
```

## Démarrage

```bash
npm start
```

L'application sera accessible sur [http://localhost:3000](http://localhost:3000)
